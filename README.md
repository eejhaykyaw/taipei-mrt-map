# Taipei MRT Tourist Map (GitHub Pages)

Static interactive map showing Taipei MRT stations + tourist spots (filters, search, nearby list).

## Deploy (GitHub Pages)
1. Create a GitHub repo
2. Upload these files to the repo root:
   - index.html
   - style.css
   - data.js
   - app.js
   - README.md
3. Go to: Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / (root)
6. Save → open the Pages URL

## Notes
- Uses OpenStreetMap tiles. Fine for personal/light use.
- If you expect heavy traffic, consider a tile provider with a free tier.
